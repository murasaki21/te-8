Technical Exercise 8:
Paper.js

Overview:

For Technical Exercise 8, you will be tasked with creating your own version of one of the Paper.js examples and hosting it on a page similar to this.

While Paper.js is most often used with Node.js, for this exercise we will be using the a simpler method by loading the file directly into our project.

Due Sunday October 24th at Midnight on Canvas

Submit links to your live site AND code.

File/Code Requirements

Your exercise will consist of an HTML page which connects to a style-sheet, the paper.js source script and your example script.:

  • The HTML page will need a canvas element to connect your script to your page. Read this page in full for details: Working With Paper.js

  • The style sheet should dictate the size of the canvas and should be used to create a cohesive color scheme.

  • The Paper.js source script an be downloaded here: Download Paper.js
    (It can also be copied from this repo.)

  • The script itself should be based on one of the examples found here: Paper.js Examples. You should be modifying the example in a significant way by doing multiple of the following:

    • Changing color and/or stroke weight
    • Changing the source imagery
    • Changing the method of interaction (mouse to keyboard or etc.)
    • Changing the speed/motion

  • Finally, you should list the changes you made and put a link to the original example and any source imagery used.
